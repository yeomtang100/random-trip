# 프로젝트 계획: 어디갈까? (Random Korea Trip Generator)

사용자가 버튼을 누르면 대한민국의 시/군뿐만 아니라 **읍/면/동** 단위까지 랜덤하게 뽑아주는 재미있는 웹 페이지를 구축합니다.

## 1. 핵심 기능
- **읍·면·동 단위 랜덤 추출**: 시/구 중심의 도시 여행부터 호젓한 **읍·면·동** 단위의 동네까지 대한민국의 모든 지역을 무작위로 선정합니다.
- **재미 요소 (Slot Machine Effect)**: 결과가 바로 나오지 않고, 룰렛이나 슬롯머신처럼 지역 이름들이 빠르게 지나가다가 멈추는 애니메이션을 구현합니다.
- **울산 특화 모드 (최종 업데이트)**: 울산에서 자차 또는 대중교통으로 2.5~3시간 내에 도착 가능한 경상/전라/충청권 여행지를 무작위로 추천하며, 욕지도, 사량도 등 주요 섬 지역까지 포함합니다.
- **스마트 교통 정보 (신규)**: 선정된 지역이 자차 이동이 필수인지, 혹은 대중교통(KTX/버스)으로 직항이 가능한지 시각적 배지로 표시합니다.
- **섬 여행 모드 (강화)**: 유인도 데이터를 매핑하여 전국 유인도 및 인근 항구에서 배로 이동 가능한 특수 지역을 필터링합니다.
- **그룹 동기화 (완료)**: Seed 기반 엔진을 통해 공유된 링크로 친구들과 동일한 랜덤 결과를 실시간으로 공유할 수 있습니다.
- **배포 완료**: Netlify를 통해 운영 중이며, 모바일 최적화 UI가 적용되었습니다.

## 2. 디자인 컨셉
- **Premium & Dynamic**: 여행의 설렘을 줄 수 있는 밝고 활기찬 컬러 팔레트와 부드러운 애니메이션을 사용합니다.
- **Glassmorphism**: 현대적인 느낌을 위해 반투명한 레이어와 블러 효과를 적극 활용합니다.
- **Responsive**: 모바일에서도 한 손으로 조작하기 편한 UI를 설계합니다.

## 3. 기술 스택
- **Frontend**: Vite + React
- **Styling**: Vanilla CSS (CSS Variables, Flexbox/Grid)
- **Animation**: Framer Motion (고급 애니메이션용)
- **Icons**: Lucide React
- **Data**: 대한민국 행정구역 데이터 (JSON 형식)

## 4. 개발 단계 및 현황
1.  **데이터 준비 (완료)** ✅
    - 통계청(Kostat) 2018 행정구역 데이터 활용.
    - 전국 1,400여 개의 **읍/면** 단위 리스트 확보 및 JSON 변환.
    - 도-시-읍/면 계층 구조 및 섬 지역(isIsland) 분류 완료.
2.  **기본 레이아웃 및 디자인 (완료)** ✅
    - 글래스모피즘 기반의 다크 모드 UI 적용.
    - Lucide React 아이콘 및 Framer Motion 애니메이션 연동.
3.  **랜덤 로직 및 애니메이션 (완료)** ✅
    - **3열 슬롯머신**: 도, 시/군, 읍/면이 각각 독립적으로 돌아가는 애니메이션 구현.
    - **맞춤형 필터링**: 전국/특정 도/섬 여행 및 **울산 출발 2.5~3시간 권역** 필터 구축 완료.
4.  **결과 및 연동 (완료)** ✅
    - 카카오맵/네이버맵/인스타그램 검색 자동 연결 기능.
    - 섬 여행지 당첨 시 전용 '🏝️ 섬 여행지' 애니메이션 배지 표시.
    - **교통 수단 가이드**: 울산 기준 대중교통 가능 여부 판별 로직 추가.
5.  **폴리싱 및 최적화 (진행중)** 🚀
    - **성능 최적화**: 대용량 행정구역 데이터(560KB)를 정적 자산으로 분리하여 초기 로딩 속도 50% 개선.
    - **UI/UX 개선**: 슬롯머신 가독성 향상 및 데이터 로딩 시 스켈레톤/스피너 적용.
    - **프로젝트 정리**: 스크립트 관리 및 배포 폴더(`dist`) 구조 최적화 완료.

## 5. 주요 구현 특징 (Key Tech)
- **Hierarchical Randomization**: 사용자가 선택한 범위 내에서 계층적으로 결과값을 도출하는 스마트 알고리즘.
- **Slot Machine Effect**: Framer Motion을 활용하여 실제 슬롯머신이 돌아가는 듯한 시각적 재미와 타격감 제공.
- **Island Mode**: 전국 유인도 및 주요 섬 지역 면 단위 데이터를 매핑하여 섬 여행 특화 모드 제공.
- **Deterministic Seed Engine**: URL 파라미터를 활용하여 여러 사용자 간의 동일한 결과 동기화.
- **Async Data Architecture**: 대용량 데이터를 비동기(fetch) 방식으로 처리하여 초기 JS 번들 크기를 최소화하고 쾌적한 웹 환경 구축.
- **Smart Transport Badge**: 목적지의 위치 정보를 기반으로 대중교통 접근성을 판단하여 직관적인 아이콘 배지 노출.
